SpaceIndices.jl

This package allows to automatically fetch and parse space indices.

The files supported in this version are:

FileExpiry periodInformation
DTCFILE.txt1 dayExospheric temperature variation caused by the Dst index.
fluxtable.txt1 dayF10.7 flux data (observed and adjusted).
SOLFSMY.txt1 dayIndices necessary for the JB2008 atmospheric model.

Installation

This package can be installed using:

julia> using Pkg
julia> Pkg.add("SpaceIndices")
